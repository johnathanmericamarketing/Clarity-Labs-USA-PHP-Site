5. IMAGE NAMING CONVENTIONS
═══════════════════════════════════════════════════════════════════════════════

  All product images go in:  images/products/{slug}/images/
  COA PDFs go in:            images/products/{slug}/pdf/

  FILENAME PATTERN          USED FOR
  ──────────────────────    ─────────────────────────────────────
  *800*                     Hero/main image on product page
  *220*                     Thumbnail preview (small)
  check_*                   Order modal product image (priority 1)
  *mockup*                  Order modal fallback image (priority 2)
  *mobile*                  Mobile-specific shop card image
  *COA*                     Certificate of Analysis thumbnail
  *.pdf                     COA PDF (auto-attached to emails)

  BPC-157 CURRENT FILES:
  • bpc_157_800px_v2.webp           — Hero image
  • bpc_157_small_perview_220px_v5  — Thumb preview
  • check_bpc_157.webp              — Order modal image
  • bpc-157_10mg_mockup_nobackground— Mockup (fallback)
  • bpc_157_mobile_341_220.webp     — Mobile shop card
  • ClarityLabsUSA_BPC157_COA_image — COA thumbnail
  • background_mockup.png           — Background asset
